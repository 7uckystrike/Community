## 배포 주소
https://react-board.herokuapp.com/

<br/>

## 시작가이드

### Requirements

#### client
* react @18.1.0
* react-router-dom @6.3.0
* react-redux @8.0.2
* firebase @9.8.2

#### server
* express @4.18.1
* mongoose @6.3.4

### Installation
~~~
$ git clone https://github.com/7uckystrike/community.git
$ cd community
~~~

#### front
~~~
$ npm install
$ npm start
~~~

#### back
~~~
$ npm 
$ npm 
~~~
